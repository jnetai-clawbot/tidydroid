# Changelog

## v1.0.0 - Initial Release

- Initial release of TidyDroid
- Automatic Downloads folder organization into categories (Videos, APKs, Music, Documents, ZIPs, Images, Other)
- Custom sorting rules engine with regex patterns
- Duplicate file finder (size + MD5 hash comparison)
- Large file cleaner with configurable minimum size threshold
- Scan history with per-category statistics
- Dark Material Design 3 theme with purple/cyan neon glow effects
- Room database for persistent storage of rules, scans, and stats
- Debug logging system with error codes (TD-xxx pattern)
- Rules management screen (add, edit, delete, enable/disable)
- Duplicates screen with grouped display and batch delete
- Cleaner screen with size filter chips
- Settings screen with About, Version, Check For Updates, Share, and Data Management
- GitHub Actions workflows for debug and release builds
- Keystore alias: tidydroid
